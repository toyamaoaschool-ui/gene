========================================
生成AIパスポート自習プログラム 技術要件書
========================================
作成日：2026年6月13日

----------------------------------------
1. システム概要
----------------------------------------
生成AIパスポート試験の合格を目的とした、
複数ユーザー対応のWebベース自習プログラム。
ログイン・問題演習・学習履歴管理の機能を提供する。

----------------------------------------
2. アーキテクチャ
----------------------------------------
種別         : シングルページアプリケーション（SPA）
構成         : フロントエンドのみ（サーバーサイド処理なし）
ビルドツール : なし（ビルドステップ不要）
配信方法     : 静的ファイルをWebサーバーで配信

----------------------------------------
3. フロントエンド
----------------------------------------
言語         : HTML5 / CSS3 / JavaScript（ES2022）
モジュール   : ES Modules（ネイティブ、バンドラー不使用）
フレームワーク: なし（Vanilla JS）
スタイリング : カスタムCSS（フレームワーク不使用）
レスポンシブ : 対応（モバイル・PC両対応）

----------------------------------------
4. バックエンド・インフラ
----------------------------------------
プラットフォーム : Firebase（Google）
プラン           : Spark（無料プラン）

【Firebase Authentication】
  - 認証方式 : メールアドレス＋パスワード
  - ユーザー管理 : Firebase が一元管理
  - セッション管理 : Firebase SDK が自動処理

【Cloud Firestore】
  - データベース種別 : NoSQL ドキュメント型
  - データ構造 :
      users/{uid}/sessions/{sessionId}
        - categoryId     : string  （カテゴリーID）
        - categoryName   : string  （カテゴリー名）
        - score          : number  （正解数）
        - total          : number  （出題数）
        - wrongIds       : array   （不正解問題IDリスト）
        - completedAt    : timestamp（完了日時）
  - セキュリティルール :
      認証済みユーザーが自分のデータのみ読み書き可能

----------------------------------------
5. 外部依存（CDN）
----------------------------------------
Firebase SDK v10.12.2
  - firebase-app.js
  - firebase-auth.js
  - firebase-firestore.js
読み込み元 : https://www.gstatic.com/firebasejs/

※ すべての依存をCDNで解決するため、
   npm / Node.js 等のインストールは不要。

----------------------------------------
6. ファイル構成
----------------------------------------
index.html          : エントリポイント（SPA シェル）
app.js              : アプリケーション本体
                      （認証・ルーティング・画面描画・
                        クイズロジック・Firestore操作）
questions.js        : 問題データ（60問）＋ユーティリティ関数
styles.css          : スタイルシート
firebase-config.js  : Firebase 接続設定
README.md           : セットアップ手順書
技術要件.txt        : 本ファイル

----------------------------------------
7. 問題データ仕様
----------------------------------------
総問題数     : 60問
カテゴリー数 : 5
問題形式     : 四択（単一正解）

カテゴリー一覧：
  1. 生成AIの基礎            （12問）
  2. 主要サービス・ツール    （12問）
  3. プロンプトエンジニアリング（12問）
  4. リスク・倫理            （12問）
  5. 法律・ガイドライン      （12問）

各問題の項目：
  - id          : 一意の問題ID
  - category    : カテゴリーID
  - question    : 問題文
  - options     : 選択肢（4件）
  - answer      : 正解インデックス（0〜3）
  - explanation : 解説文

出題方式 :
  - カテゴリー別（10問 / 20問 / 30問 / 全問）
  - 総合（全カテゴリーからランダム出題）
  - 出題順はランダムシャッフル

----------------------------------------
8. 主な機能
----------------------------------------
【認証】
  - ユーザー登録（メール＋パスワード＋氏名）
  - ログイン／ログアウト
  - 認証状態の永続化（Firebase 管理）

【ダッシュボード】
  - 学習セッション数・平均正解率の表示
  - カテゴリー別正解率のプログレスバー表示
  - カテゴリー選択＋問題数選択
  - 学習履歴（直近10件）の一覧表示

【クイズ】
  - 進捗バーの表示
  - 選択後の即時正誤判定
  - 正解・解説の表示
  - 1問ずつ進行

【結果画面】
  - スコア・正解率の表示
  - 不正解問題の解説一覧
  - 同カテゴリー再挑戦 / ダッシュボードへ戻る

【履歴管理】
  - クイズ完了時に Firestore へ自動保存
  - ユーザーごとに独立した履歴管理

----------------------------------------
9. 動作環境
----------------------------------------
【クライアント】
  対応ブラウザ : Chrome / Edge / Firefox / Safari
                （最新版を推奨、ES Modules 対応必須）
  対応デバイス : PC・スマートフォン・タブレット

【サーバー】
  要件 : 静的ファイルを配信できる Webサーバー
  必須 : HTTPS（Firebase Authentication の要件）
  例   : Apache / Nginx / Firebase Hosting 等

----------------------------------------
10. セキュリティ
----------------------------------------
  - 通信 : HTTPS 必須
  - 認証 : Firebase Authentication による管理
  - データアクセス : Firestore セキュリティルールにより
                     本人のデータのみアクセス可能
  - APIキー : Firebase の仕様上クライアントに含まれるが、
              Firestore ルール・認証ドメイン制限で保護

----------------------------------------
11. スケーラビリティ
----------------------------------------
現在の想定ユーザー数 : 21人
Firebase 無料枠の余裕 :
  - Firestore 読み取り : 約50,000回/日の上限に対し
                          想定使用量は約1,000回/日
  - Firestore 書き込み : 約20,000回/日の上限に対し
                          想定使用量は約200回/日
  - 有料プラン移行の目安 : 数百〜数千ユーザー規模
